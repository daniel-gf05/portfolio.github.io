# Portfolio
Portafolio dinámico, ligero y minimalista en Astro.

## Tecnologías usadas
- Astro
- Tailwind
- Typescript
- Vercel

## Observaciones sobre el desarrollo
- Almacenamiento de datos: He decidido crear una base de datos local en JSON debido a que la información gestionada por la aplicación es muy ligera.

- Distribución y organización: El código está organizado en componentes reutilizables, gracias a ello puedo hacer que mi portafolio se lo más ligero posible, en este proyecto considero que es lo ideal.

- Seguridad: Esta aplicación inicialmente recogía los datos de una conexión a una base de datos mongodb usando una API interna, pero debido a problemas de despliegue en Vercel, he decidido cambiar el modelo al comentado anteriormente. El enlace a la base de datos lo he almacenado en  un .env por seguridad.

- Mobile First: El layout en móviles se distribuye de forma correcta y limpia, una de las prioridades del desarrollo web moderno.

- ¿React || Astro?: Esta aplicación la he decidido hacer en Astro por el rendimiento, quería un portafolio lo más ligero posible y considero que con Astro lo he tenido mucho más fácil.

- IA: El uso de IA ha sido un segundo plano para combinaciones de colores agradables y modernos, además de código boilerplate. Para la lógica, componentes y estilos específicos he decidido no usar IA, ya que quiero reflejar mi estilo personal en mi portafolio.
